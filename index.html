Welcome to SCHUGER: Free Web-based School Management Software

Schuger is a web software one can use to build a full featured school (Nursery, Basic, college,university etc.) portal website. Schuger offer users, full customization with the use of themes and custom templates in addition to options for extending basic functionalities, with plugins that can be integrated seamlessly into the software.

core modules includes;

User Registration & Profile module
Result Processing module
Teachers Payroll module
Attendance Record module
Fee payment module
School diary module
Graded Quiz module
Timetable module
SMS module
Syllabus Builder
Discussion Forum etc.
